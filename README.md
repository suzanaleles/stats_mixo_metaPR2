# Statistical analyses for Larsson, Leles et al.

## Environmental structuring of mixoplankton functional types within marine protist communities: a global analysis

### How to run the code:
1) Unzip the dataset (metapr2_initial_data.zip)
2) Run script_som_V9_micro.R
3) Run script_som_V9_total.R
4) Run script_som_V4_total.R
5) Run script_som_comparison.R


#### Citation: Larsson, M.E., Leles, S.G., Mitra, A., Faure, E., Vaulot, D., and Santoferrera, L. Environmental structuring of mixoplankton functional types within marine protist communities: a global analysis. Ocean Microbiol. 2, 1 (2026). https://doi.org/10.1186/s44375-026-00007-3
